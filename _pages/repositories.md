A few tools I have built and made free for anyone to use. Issues and pull requests are welcome.

### [Overleaf Tracked-Changes MCP](https://github.com/cellis212/overleaf-tracked-changes-mcp-guide)

A local MCP server (and setup guide) for making Overleaf-native tracked changes programmatically. Handy for scripting edits or wiring an AI assistant into a live Overleaf project without giving up the accept/reject review layer.

### [Customizable Specification Curves in R](https://github.com/cellis212/Customizable-Spec-Curves-R)

Modifiable (not canned) code, built on the `specr` package, for producing highly flexible, publication-quality specification curves in R. It makes it easy to build curves across several dependent variables, and to sample when the number of specifications gets large.

### [Scrape Morningstar Annuities](https://github.com/cellis212/Scrape-Morningstar-Annuities)

Code to scrape annuity data from Morningstar's AnnuityIntel platform and process it into an analysis-ready format.
